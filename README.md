# 📊 Retail Sales Analysis using Python

## 🧾 Project Overview

This project analyzes retail sales data to understand business performance across different regions, categories, and time periods.
The analysis is performed using Python and includes data cleaning, visualization, and business insights.


## 📊 Project Highlights

• Data Cleaning and Preprocessing
• Sales and Profit Analysis
• Region-wise and Category-wise Performance
• Trend Analysis over Time
• Visualization using charts


## 🛠 Tools & Technologies

• Python
• Pandas
• Matplotlib
• Seaborn


## 📈 Key Insights

• The West region demonstrates the highest sales performance, indicating strong market demand.
• The Technology category generates the highest profit among all categories.
• Higher discount levels negatively impact profitability.
• Sales show a consistent upward trend over the years.
• Some categories generate high sales but low profit, indicating cost inefficiencies.


## 💼 Business Recommendations

• Focus on expanding high-performing regions.
• Invest more in profitable product categories.
• Optimize discount strategies to maintain profit margins.
• Improve cost management for better profitability.
